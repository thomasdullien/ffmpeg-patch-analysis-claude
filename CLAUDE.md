The goal is to analyze security reports for the FFMpeg project, and to build
patches for the code.

* The FFMpeg bug tracker is available under https://trac.ffmpeg.org/ticket/XXXXX where the XXXXX are the ticket number.

There are a number of subagents that you can use to help with this task. If you
are provided with a bug tracker ticket ID, invoke the security-bug-analyzer-agent
with this information.
